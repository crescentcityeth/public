About This Website

This is a decentralized website located on the Ethereum blockchain at CrescentCity.eth. To browse on the Ethereum network, you'll need a Web 3.0 browser, like Opera, Brave or Metamask.

This website is bridged to the traditional DNS network at https://crescentcity.eth.link with the eth.link service. Learn more about the decentralized web

You can send any ERC-20 token (like Ethereum or USDC) directly to this website from any Web 3.0 wallet.
